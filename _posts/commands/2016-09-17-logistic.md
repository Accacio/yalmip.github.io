[logistic](/command/logistic) is an implementation of the logistic function  \\(\frac{1}{1+exp^{-x}}\\)

### Syntax

````matlab
y = logistic(x)
````

### Implementation

The operator  is implemented as a [callback operator](/tutorial/nonlinearoperatorscallback).
