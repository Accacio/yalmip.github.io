[intvar] used to define decision variables with integer elements.

### Syntax

````matlab
x = intvar(n)
x = intvar(n,m,)
x = intvar(n,m,'type'
x = intvar(n,m,'type','field')
intvar x
````

### Examples

A scalar integer variable is defined with

````matlab
P = intvar(1)
````

For more examples, see [sdpvar].
