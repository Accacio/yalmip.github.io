[floor] overloads nchoosek operator with a mixed-integer model

### Syntax

````matlab
y = nchoosek(X,k)
````

### Comments

Note that only **X** can be a decision variable, **k** has to be a constant integer.

The operator is implemented using a [mixed-integer model].
